Preliminary Math

Machine learning relies heavily on a strong foundation in mathematics, specifically calculus, linear algebra, and statistics. While it is possible to learn machine learning without an extensive math background, understanding the underlying concepts and algorithms is much more challenging without it.

There is an abundance of materials covering these topics in depth, as they are commonly taught in university Math and Computer Science programs. This page highlights the key topics that should be understood before diving into machine learning. It is perfectly fine to learn these on the go, but having at least a surface-level understanding of these topics will make learning machine learning much easier.

Linear Algebra
  • Vectors
  • Matrices
  • Matrix Multiplication
  • Dot Product
  • Euclidean Norm
Calculus
  • Derivatives
  • Partial Derivatives
  • Chain Rule
  • Convexity
  • Gradients
Statistics
  • Independence
  • Conditional Probability
  • Probability Distributions
  • Binomial Distribution
  • Normal Distribution
Python
  • Variables
  • Data Types
  • Functions
  • Flow Control (if statements, loops)
  • Data Structures (lists, tuples, dictionaries)
  • File I/O
  • NumPy
  • Basic Object-Oriented Programming (OOP)
