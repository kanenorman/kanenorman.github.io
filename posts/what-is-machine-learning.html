Truthfully, it depends on who you ask. To some, it's a mysterious black box that can magically answer any question. For academics, it's advanced math that can mimic human decision-making. For skeptics, it signals the end of the world, with Terminator robots coming to end humanity!

Regardless of how you define it, at its core, machine learning revolves around creating algorithms that are designed to identify patterns in data and make predictions based on those patterns. [1] (Not too different from how humans learn, right?)

Types of Machine Learning

Different kinds of datasets and problems lend themselves to different types of learning [2]

Supervised Learning: Algorithms are trained on labeled data where the desired output is known. For example, in a spam email classifier, the algorithm learns from emails labeled as "spam" or "not spam" to predict the label of new, unlabeled emails.

Unsupervised Learning: Involves algorithms learning patterns from data without explicit labels. The goal is to explore the structure of the data, such as clustering similar data points together or reducing the dimensionality of the data.

Reinforcement Learning: Is about training agents to make sequences of decisions by interacting with an environment. The agent learns to achieve a certain goal through trial and error, receiving feedback in the form of rewards or penalties.

We will primarily focus on supervised learning, where the emphasis is on training models with labeled data to make accurate predictions or classifications.

Supervised Learning: A Gentle Example

Imagine you have collected 50 measurements for height and weight. You want to build a machine learning model to predict a subject's weight given their height.

Observation Height (cm) Weight (kg)
1 150 50
2 160 55
3 170 60
... ... ...
48 194 84
49 196 86
50 198 88

When we plot this data, a clear pattern emerges: as height increases, weight increases in a roughly linear manner. This fundamental observation underscores the core principal of machine learning, identify patterns in data and make predictions based on those patterns [1]

Given our labeled data, we can employ a supervised learning algorithm to understand and model this linear pattern effectively.

The question now is, how does the machine learn this linear pattern? We know the model needs to learn a linear equation in the form \( y = B_0 + B_1x \). But how does the model find the optimal parameters \( B_0 \) and \( B_1 \)?

Without diving too far into the details just yet, the algorithm starts with random initial values and iteratively adjusts the parameters to minimize the error between the model's predictions and the actual labeled data.

By learning this linear pattern from our data, we can train a model that predicts weight based on height.. The resulting line is known as a linear regression model.
